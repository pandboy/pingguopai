# pingguopai
## 简介
pingguopai是学习使用spring boot 如何结合使用spring mvc, mybatis, druid数据源的项目，是基于https://github.com/lihengming/spring-boot-api-project-seed/ 项目开发调整的
## 新特性
- 添加model基类，保存更新记录更新时间
- 添加分页接口
- mapper中添加PK泛型，便于适应现有项目架构
- 基于jwt认证体系
- 内存缓存
## 待加特性
